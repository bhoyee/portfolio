# Welcome

This blog is **static** and powered by local Markdown files in `src/content/blog`.

## How it works

- The blog list reads all `*.md` files at build time.
- Each file can include simple frontmatter (`title`, `date`, `tags`, etc.).
- The post body is rendered with `react-markdown`.
